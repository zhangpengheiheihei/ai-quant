pos_prev[1:] = pos[:-1]  # 使用前一期持仓计算当期收益

## 五、主要发现与结论

### 5.1 策略特性

| 特性 | 说明 |
|------|------|
| 信号性质 | 右侧交易，滞后确认 |
| 核心价值 | 风险控制（通过离场截断亏损） |
| 局限性 | 震荡市中频繁交叉产生磨损 |

### 5.2 适用场景

✅ **平滑趋势市场**：趋势方向明确、回调不频繁的标的  
✅ **中长线持仓**：10/30、20/60等较长周期过滤噪声  
✅ **组合风控**：作为纪律化离场工具，规避"扛单"人性弱点

### 5.3 不适用场景

❌ **震荡市**：价格区间来回，均线频繁交叉 → 磨损严重  
❌ **高交易成本环境**：短周期参数交易频繁，手续费侵蚀收益  
❌ **突发拐点**：均线天然滞后，无法在尖顶/尖底及时反应

### 5.4 应用建议

1. 参数没有"最优"，只有"最适配"，需结合标的特性选择
2. 务必与买入持有基准对比，只有风险调整后收益真正改善才有实盘价值
3. 单一双均线策略信号较粗糙，实战中常配合成交量、MACD、布林带等过滤器
